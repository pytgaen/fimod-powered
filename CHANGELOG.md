# Changelog

## 0.1.0 — 2026-03-29

Initial release with 6 molds:

- **dockerfile** — Generate production Dockerfiles from JSON/YAML descriptors
- **html_report** — Convert JSON/CSV data to standalone HTML reports
- **poetry_migrate** — Migrate Poetry pyproject.toml to Poetry 2 or uv
- **download** — Download files from URLs
- **gh_latest** — Get latest GitHub release tag or download URL
- **skylos_to_gitlab** — Convert Skylos dead code reports to GitLab Code Quality format
